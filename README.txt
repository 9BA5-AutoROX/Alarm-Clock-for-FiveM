# ⏱️ Gaming Overlay Timer & Alarm (v0.1.1 Beta)
** แก้ไขเวลาเก้าอี้ดนตรีให้ตรง และ เพิ่มคู่มือการใช้งาน Readme.txt**

โปรแกรมนาฬิกาจับเวลาและแจ้งเตือนแบบ Overlay สำหรับการเล่นเกม (ออกแบบมาเพื่อ FiveM และเกมทั่วไป)

## ✨ คุณสมบัติหลัก
- **Always-on-Top**: แสดงผลทับหน้าจอเกมตลอดเวลา
- **Click-Through Lock**: ล็อคหน้าจอเพื่อไม่ให้เมาส์ไปโดนนาฬิกาขณะเล่นเกม (กด `Ctrl + Shift + L`)
- **Global Hotkeys**: เรียกใช้งาน Preset ได้ทันทีแม้พับหน้าจออยู่
- **Preset Manager**: จัดการการนับถอยหลัง (Countdown) และการแจ้งเตือนตามเวลาจริง (Real-time Alarm) ได้เอง

---

## ⌨️ ปุ่มลัด (Hotkeys)
| คำสั่ง | ปุ่ม (Hotkey) | รายละเอียด |
|---|---|---|
| **Lock/Unlock** | `Ctrl + Shift + L` | สลับโหมดคลิกทะลุ (Locked = สีแดง) |
| **Preset 1-9** | `Ctrl + Shift + 1-9` | เรียกใช้ Countdown Preset ลำดับที่ 1-9 ทันที |

---

## 🚀 วิธีเริ่มใช้งาน
1. เข้าไปที่โฟลเดอร์ `AlarmClock_v0.1.1/`
2. ดับเบิลคลิกที่ **`AlarmClock_v0.1.1.exe`**
3. **การตั้งค่าตำแหน่ง**: ลากหน้าต่างไปวางในจุดที่ต้องการ (ระบบจะจำตำแหน่งเดิมเมื่อเปิดเครื่องใหม่)
4. **คลิกขวาที่หน้าต่าง** เพื่อเปิดเมนูตั้งค่าและ Preset Manager

---

## 📁 โครงสร้างโฟลเดอร์สำหรับผู้ใช้
- `AlarmClock_v0.1.1.exe`: ตัวโปรแกรมหลัก
- `presets/`: ไฟล์ `.json` สำหรับแก้ไขค่าล่วงหน้า (แก้ไขในโปรแกรมสะดวกกว่า)
- `settings/`: เก็บตำแหน่งหน้าต่าง (ไม่ต้องยุ่งกับไฟล์นี้)
