# ⏱️ Gaming Overlay Timer & Alarm (v0.2 Beta)
** เพิ่มหน้าต่างจับเวลาแยกสำหรับ เห็ด และ วัว **

โปรแกรมนาฬิกาจับเวลาและแจ้งเตือนแบบ Overlay สำหรับการเล่นเกม (ออกแบบมาเพื่อ FiveM และเกมทั่วไป)

## ✨ คุณสมบัติหลัก
- **Always-on-Top**: แสดงผลทับหน้าจอเกมตลอดเวลา
- **Click-Through Lock**: ล็อคหน้าจอเพื่อไม่ให้เมาส์ไปโดนนาฬิกาขณะเล่นเกม (กด `Ctrl + Shift + L`)
- **Global Hotkeys**: เรียกใช้งาน Preset ได้ทันทีแม้พับหน้าจออยู่
- **Preset Manager**: จัดการการนับถอยหลัง (Countdown) และการแจ้งเตือนตามเวลาจริง (Real-time Alarm) ได้เอง

---

## ⌨️ ปุ่มลัด (Hotkeys)
| คำสั่ง | ปุ่ม (Hotkey) | รายละเอียด |
|---|---|---|
| **Lock/Unlock** | `Ctrl + Shift + L` | สลับโหมดคลิกทะลุ (Locked = สีแดง) |
| **Main Timer 1-9** | `Ctrl + Shift + 1-9` | เรียกใช้ Countdown ลำดับที่ 1-9 (ชุดบน) ทันที |
| **Feed Mushroom** | `Ctrl + Shift + M` | เริ่มนับถอยหลังให้อาหารเห็ด 1 ชั่วโมง (ชุดซ้ายล่าง) |
| **Cow (Normal)** | `Ctrl + Shift + C` | เริ่มนับถอยหลังให้อาหารวัว 5 นาที (ชุดขวาล่าง) |
| **Cow (VIP)** | `Ctrl + Shift + V` | เริ่มนับถอยหลังรอเก็บเนื้อวัว 25 นาที (ชุดขวาล่าง) |

## 🚀 วิธีเริ่มใช้งาน
1. เข้าไปที่โฟลเดอร์ `AlarmClock_v0.2/`
2. ดับเบิลคลิกที่ **`AlarmClock_v0.2.exe`**
3. **การตั้งค่าตำแหน่ง**: ลากหน้าต่างไปวางในจุดที่ต้องการ (ระบบจะจำตำแหน่งเดิมเมื่อเปิดเครื่องใหม่)
4. **คลิกขวาที่หน้าต่าง** เพื่อเปิดเมนูตั้งค่าและ Preset Manager

---

## 📁 โครงสร้างโฟลเดอร์สำหรับผู้ใช้
- `AlarmClock_v0.2.exe`: ตัวโปรแกรมหลัก
- `presets/`: ไฟล์ `.json` สำหรับแก้ไขค่าล่วงหน้า (แก้ไขในโปรแกรมสะดวกกว่า)
- `settings/`: เก็บตำแหน่งหน้าต่าง (ไม่ต้องยุ่งกับไฟล์นี้)
- `sounds/`: โฟลเดอร์สำหรับใส่ไฟล์เสียงแจ้งเตือนแบบ `.wav` (ถ้าระบบหาไฟล์เสียงไม่เจอ จะไม่มีเสียงเตือนแต่รันได้ปกติ)
  - `feed_mushroom.wav` (เสียงเห็ดเสร็จ)
  - `cow_normal.wav` (เสียงวัว 5 นาทีเสร็จ)
  - `cow_vip.wav` (เสียงวัว 25 นาทีเสร็จ)
  - `warning_5m.wav`, `warning_3m.wav`, `warning_1m.wav` (เสียงเตือนล่วงหน้า)
  - `default_complete.wav` (เสียงเตือน Default ทั่วไป)
